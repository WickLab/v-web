# Vidula Portfolio (Next.js + Tailwind + Framer Motion)

## Run
npm install
npm run dev

## Build
npm run build
npm run start

## Customize
Edit:
- components/Home/Hero.tsx
- components/Home/FeaturedProjects.tsx
- app/portfolio/page.tsx (project data)
- app/achievements/page.tsx (certifications + timeline)
- app/contact/page.tsx (contact info)

Put your resume at: public/resume.pdf
Put project images at: public/projects/*
